FFT Ranking Dashboard V2.2 - Keep UI + Dynamic Month

What changed from V2:
- The dashboard design/layout remains the same as V2.
- Only Year and Month dropdown logic was improved.
- Year/Month options are now read automatically from ranking.csv.
- Supports full English month names, short English month names, numeric months, and Thai month names.

How to use:
1. Open FFT Scoring Log.xlsx.
2. Update training scores.
3. Save As CSV UTF-8 and overwrite ranking.csv in this folder.
4. Open/start the dashboard.
5. Press Refresh Data or Ctrl+F5 in the browser.

Do not rename:
- ranking.csv
- photos folder
- index.html / script.js / style.css

You may rename the main folder.
